🔍 How To Use

  • Visit any source above, open a document (PDF or HTML)
  • Select All → Copy the text content
  • Paste into the Analyze tab's text box
  • Hit ⚡ Extract Intelligence
  • Switch to the Feed tab to read your results
  • Hit ⊕ Save in the header to save the feed for later
