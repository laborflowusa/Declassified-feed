# DECLASSIFIED // Intelligence Feed

**AI-Powered Declassified Document Analyzer** — A Progressive Web App

Drop any declassified government document (PDF or TXT). Instantly extract the most shocking revelations into sharp, Twitter-style intelligence cards.

---

### 🔴 Live App
https://lively-churros-0fd3db.netlify.app

### 📱 Features
- Drag & drop PDFs or TXT files (automatic text extraction)
- Optional OCR enhancement for scanned or redacted documents
- AI analysis powered by Groq Llama 3.3 70B
- Beautiful Twitter/X-style intelligence feed with shock levels
- Save feeds locally (up to 20)
- One-tap "Post to X" sharing
- Fully installable PWA (works offline for saved feeds)
- Mobile + Desktop friendly

### 🛠 How It Works
1. Drop a PDF or paste document text
2. Click **Extract Intelligence**
3. For scanned/redacted PDFs, use the **Enhance with OCR** button if needed
4. Browse the resulting intelligence feed

### 📡 Recommended Sources
- [CIA Reading Room](https://www.cia.gov/readingroom)
- [FBI Vault](https://vault.fbi.gov)
- [National Archives JFK Records](https://www.archives.gov/research/jfk)
- [NSA FOIA Reading Room](https://www.nsa.gov/resources/foia/)

### ⚠️ Notes
- OCR is an optional tool for scanned/image-based PDFs. Redacted black bars cannot be recovered.
- All analysis is generated from publicly available declassified documents.
